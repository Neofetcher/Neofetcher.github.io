AWS Cloud Security Work

devansh@cloud-portfolio:~$ aws describe-work --profile devansh

Most of my cloud work is on AWS, focusing on:

  • Designing segmented VPCs for public, private, and management subnets.
  • Using security groups and routing rules to strictly control east-west and north-south traffic.
  • Applying least-privilege IAM for users, roles, and services, avoiding broad wildcard policies.
  • Enabling logging and monitoring (flow logs, CloudTrail, basic alerting) as part of the baseline.
